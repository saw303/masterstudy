\section{Frage 8 - Materiell oder Immaterielle Anreize}
Welche Art von nachhaltiger Mitarbeiterbindung (Anreize) bevorzugen Sie? \textit{(Nur eine Auswahl möglich)}
\begin{itemize}
	\item Materiell (direkte Zahlungen, Lohnerhöhung, usw. )
	\item Immateriell (indirekte Zahlungen wie Weiterbildung, flexible Arbeitszeiten, usw.)
\end{itemize}

\section{Frage 9 - Materielle Anreiz}
Wieso bevorzugen Sie materielle Anreize? \textit{(Nur eine Auswahl möglich)}
\begin{itemize}
	\item Ich bin der Überzeugung dass mir eine direkte Zahlung in jeglicher Form einen Mehrwert bringt
	\item Ich kann mit immateriellen Anreizen nichts anfangen
	\item Das Unternehmen bietet mir nur die Möglichkeit des materiellen Anreizes
\end{itemize}

\section{Frage 10 - Materielle Favoriten}

\begin{itemize}
	\item Lohnerhöhung
	\item Lohn mit variablem, ergebnisbezogenem Anteil
	\item Bonus
\end{itemize}

\section{Frage 11 - Immaterielle Anreize}
Wieso bevorzugen Sie immaterielle Anreize? \textit{(Nur eine Auswahl möglich)}
\begin{itemize}
	\item Ich bin der Überzeugung dass mir eine indirekte Zahlung in jeglicher Form einen Mehrwert bringt
	\item Ich kann mit materiellen Anreizen nichts anfangen
	\item Das Unternehmen bietet mir nur die Möglichkeit des immateriellen Anreizes
\end{itemize}
